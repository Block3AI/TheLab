# Basic Investment Calculator

A small, browser-based calculator for estimating the future value of an investment using an initial deposit, monthly contributions, a growth period, and an annual rate of return.

## Run locally

1. Clone or download this repository.
2. Open `index.html` in your preferred web browser.

No build step or dependencies are required.

## How it works

Enter your initial investment, monthly contribution, and number of years. Use the `+` and `-` controls to set the annual return rate, then select **Calculate**. The calculator applies the annual return and yearly total of monthly contributions for each year in the selected period.

## Project files

- `index.html` — page structure and calculator form
- `style.css` — calculator styling
- `index.js` — calculation, rate controls, and reset behavior
