# Week 2: Data and Modules

## Focus

Learn to model information with arrays and objects, transform data with array methods, and organize code across modules.

## Build

Create a crypto watchlist tool that can add assets, calculate a portfolio total, filter assets by criteria, and display a readable summary.

## Topics

- Arrays and objects
- `map`, `filter`, `find`, and `reduce`
- Destructuring and spread syntax
- Importing and exporting modules
- Basic input validation

## Done when

- Your watchlist data lives in a separate module.
- Your app calculates totals without mutating the original data.
- You can explain when to use `map`, `filter`, and `reduce`.
