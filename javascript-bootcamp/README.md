# JavaScript Bootcamp

A six-week, build-first JavaScript learning path for the Block3AI Lab. Each week contains a short roadmap, practice goals, and a deliverable. Start with Week 1 and move forward only after you can explain and run what you built.

## Prerequisites

- Node.js 18 or newer
- A code editor such as Cursor
- Git and a GitHub account

Check Node is installed:

```bash
node --version
```

## Six-week roadmap

| Week | Focus | Deliverable |
| --- | --- | --- |
| 1 | JavaScript fundamentals | Command-line investment calculator |
| 2 | Arrays, objects, and modules | Crypto watchlist data tool |
| 3 | HTML, CSS, DOM, and events | Interactive browser dashboard |
| 4 | Async JavaScript and APIs | Market-data viewer with loading and error states |
| 5 | Node.js and backend basics | Small JSON API for a portfolio |
| 6 | Capstone and deployment | Polished Web3 or AI mini-project |

## Folder structure

```text
javascript-bootcamp/
├── week-01-fundamentals/     # Start here
├── week-02-data-and-modules/
├── week-03-dom-and-events/
├── week-04-async-and-apis/
├── week-05-node-and-backend/
└── week-06-capstone/
```

## How to use this repository

1. Read the week’s `README.md`.
2. Complete the practice tasks before extending the project.
3. Keep notes on what worked, what broke, and what you learned.
4. Commit your work at the end of each study session.

## Start Week 1

```bash
cd week-01-fundamentals/starter
node index.js
```

The starter runs without installing packages. Complete the `TODO` items in `exercises.js` and `calculator.js`, then rerun the command to check your progress.
