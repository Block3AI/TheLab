# Week 4: Async JavaScript and APIs

## Focus

Fetch remote data safely and present loading, success, and failure states in the UI.

## Build

Add live or mock market pricing to the Week 3 dashboard.

## Topics

- Promises and `async`/`await`
- `fetch` and JSON
- HTTP status handling
- Loading and error states
- Environment variables and keeping API keys private

## Done when

- The UI clearly shows that data is loading.
- API failures show a helpful, recoverable error message.
- No secret value is committed to Git.
