# Week 3: DOM and Events

## Focus

Use JavaScript in the browser to select elements, respond to events, update the DOM, and manage simple UI state.

## Build

Turn the crypto watchlist into an interactive browser dashboard with an add-asset form, a rendered asset list, and a running total.

## Topics

- HTML form controls
- DOM selection and element creation
- Click, submit, and input events
- Rendering from state
- Separating data logic from UI logic

## Done when

- Adding an asset updates the page without a refresh.
- Invalid form input displays a useful message.
- Resetting the form does not erase existing assets.
