# Week 5: Node.js and Backend Basics

## Focus

Move JavaScript logic to a server, create a small HTTP API, and work with structured request and response data.

## Build

Create a small portfolio API with endpoints to list assets, add an asset, and return a portfolio summary.

## Topics

- Node.js runtime and npm scripts
- HTTP methods and status codes
- JSON request and response bodies
- Route handlers
- Validation and error responses

## Done when

- `GET` and `POST` requests work locally.
- Invalid requests receive a clear 4xx response.
- Your API has a short README with run instructions.
