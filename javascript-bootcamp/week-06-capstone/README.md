# Week 6: Capstone and Deployment

## Focus

Plan, build, document, and publish one small project that combines the skills from the bootcamp.

## Choose a project

- Crypto portfolio tracker
- Wallet activity viewer
- AI-assisted research notes tool
- Token-gated checklist prototype
- Investment scenario calculator

## Required scope

- A clear user problem and one-sentence project goal
- A responsive, usable interface
- At least one data flow: user input, API data, or persisted local data
- Error and empty states
- A complete README with setup, features, and screenshots
- A deployed version or a documented local demo

## Finish line

Demo the project, open a pull request, and write a short retrospective: what you built, what you learned, and what you would improve next.
